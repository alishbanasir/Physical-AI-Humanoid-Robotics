# Example 03: Batch URDF to USD Conversion

## Overview

Automates conversion of multiple URDF files to USD format using batch processing. Useful for converting entire robot model libraries.

## Usage

```bash
./python.sh main.py --input-dir /path/to/urdfs/ --output-dir /path/to/usds/
```

## Learning Objectives

- Batch process multiple URDF files
- Handle conversion errors gracefully
- Generate conversion reports
