# Example 04: Custom USD Scene Creation

## Overview

Creates custom simulation environments with ground planes, obstacles (boxes, cylinders), and imported robots. Demonstrates USD scene composition.

## Usage

```bash
./python.sh main.py --robot /path/to/humanoid.usd --output custom_scene.usd
```
